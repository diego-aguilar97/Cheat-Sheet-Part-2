HTML Reference Cheat Sheet

Common HTML Elements

Tag Name Description
<h1> Heading 1 The largest in the heading hierarchy
<p> Paragraph Surrounds the words that make up your paragraphs with an opening and closing tag
<div> Div Short for division, it serves as a container that divides the page into sections
<body> Body Only content inside the opening and closing tags an be displayed to the screen.
<head> Heading Comes before body element and contains information about the page such as the title in your open tab.

HTML Attributes

Attribute Name Description
<div id="example">Example</div> id Used to specify a unique id for an HTML element, which cannot be shared across multiple elements within the same HTML document.
<div class="example">Example</div> class The class attribute is used for assigning styles to elements. Multiple classes can be assigned to a single element in a space-separated list.
<element src="url"> Source The src attribute specifies the location of a digital source, which is typically a URL or a file path. It is required for HTML elements like elements so that they can appear on a website or application.
